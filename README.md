##  Pages
1. **index.html** - Page d'accueil avec films tendances
2. **search.html** - Recherche en temps réel
3. **movie.html** - Détails d'un film
4. **details.html** - Informations supplémentaires

##  Fonctionnalités
-  Affichage des films populaires
-  Recherche instantanée
-  Fiche détaillée des films
-  Design responsive
-  API OMDb intégrée

##  Lien GitHub
**Dépôt principal** : https://github.com/junior-Christophe/cinema-village-projet

## Installation
1. Cloner : `git clone https://github.com/junior-Christophe/cinema-village-projet.git`
2. Ouvrir `index.html` dans un navigateur

##  Configuration API
Ajouter votre clé OMDb dans le fichier de configuration.
